## 🎯 Ementa e Objetivos

A disciplina apresenta temas avançados de Banco de Dados, introduzindo conceitos de **Ciência de Dados**, **Big Data** e **Business Intelligence**.

Ao longo do semestre, o estudante deverá:

- compreender dados estruturados e não estruturados;
- aplicar técnicas de visualização, mineração, extração e análise de dados;
- experimentar algoritmos e abordagens práticas para análise de dados;
- selecionar tecnologias adequadas para diferentes cenários de software e dados.

## 🧠 Conteúdos Abordados

- Introdução a Big Data e dados estruturados vs. não estruturados
- KDD: obtenção, limpeza, mineração, extração e análise de dados
- Aplicação de algoritmos e experimentos de Machine Learning
- Visualização de dados e aplicações de BI
- Experimentos de análise de dados
- Bancos de dados pós-SQL, não convencionais e distribuídos

## 🧪 Ferramentas e Tecnologias

- Apache Superset e Microsoft Power BI (Professional)
- Linguagens e bibliotecas para análise de dados (conforme aula/lab)
- VS Code e/ou PyCharm/IntelliJ
- Serviços em nuvem para manipulação de grande volume de dados

## 🧑‍🏫 Metodologia

As aulas são conduzidas em **laboratório**, integrando teoria e prática no mesmo ambiente. A experimentação prática guiada pelo docente é parte central da aprendizagem na disciplina.

## 📝 Avaliação (resumo)

- 1 atividade prática individual (`AtivInd`), valendo até 10 pontos
- 1 atividade prática em equipe (`AtivEq`) em formato de mini-projeto
- Projeto Integrador (`PI`) quando aplicável

Fórmula da nota final:

`NF = (AtivInd * 0,5) + (AtivEq * 0,4) + (PI * 0,1)`

Recuperação:

- Estudantes com média final inferior a 5,0 realizam avaliação teórica substitutiva, abrangendo todo o conteúdo do semestre.
- A avaliação substitutiva troca a menor nota entre `AtivInd` e `AtivEq`.
